# 🧁 Budines Lab

Este directorio contiene el desarrollo iterativo de recetas de budines.

## 🎯 Objetivo
Desarrollar recetas optimizadas mediante iteraciones controladas, registrando cambios y resultados.

## 🧪 Metodología
- Cambiar UNA variable por iteración
- Registrar resultados
- Comparar con versiones anteriores

## 📂 Estructura

- `receta_base.md` → receta inicial (control)
- `iteraciones/` → pruebas realizadas
- `plantillas/` → templates reutilizables

## 📈 Criterios de evaluación

- Altura
- Miga (aireada / compacta)
- Humedad
- Sabor
- Duración

Evolucionar hacia un budín de banana y nuez optimizado
